## [1629] 곱셈

https://www.acmicpc.net/problem/1629

**문제 유형**

재귀 (분할 정복 + 모듈러 연산)

<br>


**풀이 방법 도출**

```
// 재귀 내려가기 (5, 8, 7)

pow(5, 8)  →  half = pow(5, 4)
                      half = pow(5, 2)
                                half = pow(5, 1)
                                          ↓
                                     return 5%7 = 5
```

```
// 올라오기

pow(5, 1) = 5%7 = 5

pow(5, 2):  half=5,  (5*5) % 7 = 25%7 = 4

pow(5, 4):  half=4,  (4*4) % 7 = 16%7 = 2

pow(5, 8):  half=2,  (2*2) % 7 = 4%7 = 4
```

1. b가 1이 될 때까지 b를 절반씩 줄이며 재귀 호출

2. 돌아오면서 half * half로 제곱 -> 지수를 두 배씩 복원

3. b가 홀수이면 정수 나눗셈으로 1이 잘렸으므로 a를 한 번 더 곱해서 보정

<br>

**시간 복잡도**

O(log b)
