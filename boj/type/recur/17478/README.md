## [17478] 재귀함수가 뭔가요?

https://www.acmicpc.net/problem/17478

**문제 유형**

재귀

<br>


**풀이 방법 도출**

```
chatbot(0)
  → "재귀함수가 뭔가요?"
  → "잘 들어보게..."
  → chatbot(1)          <- 첫번째 재귀
      → "재귀함수가 뭔가요?"
      → "잘 들어보게..."
      → chatbot(2)       <- 두번째 재귀
          ...
            → (depth==n) "재귀함수가 뭔가요?"
                         "재귀함수는 자기 자신을..."
                         "라고 답변하였지."   ← 23줄 (base case)
                         return
          ← 돌아와서
          "라고 답변하였지."   <- 34줄 (depth==n-1 수준)
      ← 돌아와서
      "라고 답변하였지."       <- 34줄 (depth==n-2 수준)
  ← 돌아와서
  "라고 답변하였지."           <- 34줄 (depth==0 수준)
```

1. chatbot(depth, under)를 재귀 호출하며 depth가 n이 되면 베이스 케이스 출력 후 리턴

2. 매 호출마다 들여쓰기(under)에 ____를 추가해 중첩 깊이를 표현

3. 재귀 호출 후 돌아오면서 "라고 답변하였지."를 출력해 역순으로 마무리

<br>

**시간 복잡도**

O(N)
